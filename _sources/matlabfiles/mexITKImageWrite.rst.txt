mexITKImageWrite
================

.. code-block:: none

  % mexITKImageWrite: write Matlab image matrix to any image format supported by ITK using itk::Image and itk::VectorImage.
  %
  % Usage:
  %   mexITKImageWrite('filename', image, [params])
  %
  % INPUT
  %  filename   :     file name to save. 
  %  image      :     4D matlab matrix
  %
  %  params.origin            :  origin in image 
  %  params.spacing           :  spacing in image 
  %  params.vectorImage       :  If it is true, save 3D itk::VectorImage, otherwise save 4D itk::Image (default).
