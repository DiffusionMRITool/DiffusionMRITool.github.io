SPFToODF
========

.. code-block:: none

  
  USAGE: 
  
     SPFToODF  [--returnparameterfile <std::string>]
               [--processinformationaddress <std::string>] [--xml] [--echo]
               [--debug] [--md0 <double>] [--tau <double>] [--nt <int>]
               [--mask <std::string>] [--mdImage <std::string>] [--scale
               <double>] [--bMax <double>] [--odfOrder <0|2|-1>] [--ra <int>]
               [--sh <int>] [--] [--version] [-h] <std::string>
               <std::string>
  
  
  Where: 
  
     --returnparameterfile <std::string>
       Filename in which to write simple return parameters (int, float,
       int-vector, etc.) as opposed to bulk return parameters (image,
       geometry, transform, measurement, table).
  
     --processinformationaddress <std::string>
       Address of a structure to store process information (progress, abort,
       etc.). (value: 0)
  
     --xml
       Produce xml description of command line arguments (value: 0)
  
     --echo
       Echo the command line arguments (value: 0)
  
     --debug
       debug (value: 0)
  
     --md0 <double>
       Typical MD value. (value: 0.0007)
  
     --tau <double>
       Tau value. The default is calculated based on 4*pi*pi*tau=1. (value:
       0.0253303)
  
     --nt <int>
       Number of threads. If it not positive, then the default number for
       multiple threads is used. (value: -1)
  
     --mask <std::string>
       Mask file.
  
     --mdImage <std::string>
       Mean diffusivity Image for adaptive scale.
  
     --scale <double>
       Scale for SPF basis. If it is negative, the default value based on
       typical MD is used. (value: -1)
  
     --bMax <double>
       Maximal b value for disk integral. If it is -1, it is a plan integral.
       (value: -1)
  
     --odfOrder <0|2|-1>
       ODF order. 2 for ODF with solid angle, 0 for Tuch ODF. -1 for ODF
       estimated based on Funk-Radon transform. (value: 2)
  
     --ra <int>
       Rank for radial basis. (value: 1)
  
     --sh <int>
       Rank for SH basis. (value: 4)
  
     --,  --ignore_rest
       Ignores the rest of the labeled arguments following this flag.
  
     --version
       Displays version information and exits.
  
     -h,  --help
       Displays usage information and exits.
  
     <std::string>
       (required)  Input Image File represented by SPF basis.
  
     <std::string>
       (required)  Output Image File (ODF).
  
  
     Description: Convert SPF coefficients to ODFs (represented by SH basis
     or discrete samples).
  
     Examples: 
  
     SPFToODF signalSPF.nii.gz odf.nii.gz --sh 8 --ra 4 --mdImage MD.nii.gz 
  
     Reference: 
  
     Jian Cheng, Aurobrata Ghosh, Rachid Deriche, Tianzi Jiang, 'Model-Free,
     Regularized, Fast, and Robust Analytical Orientation Distribution
     Function Estimation', Medical Image Computing and Computer-Assisted
     Intervention (MICCAI'10), vol. 6361, pp. 648–656, sep, 2010.

     and Analytical EAP Reconstruction via Spherical Polar Fourier Diffusion
     MRI', Medical Image Computing and Computer-Assisted Intervention
     (MICCAI'10), vol. 6361, pp. 590–597, sep, 2010.
